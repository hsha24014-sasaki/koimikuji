
恋みくじつよつよ

ボタンをクリックするだけで、今日の恋愛運を占うことができるシンプルなウェブアプリケーションです。

## 特徴

- **シンプル操作**: 「おみくじを引く」ボタンを押すだけで結果が表示されます。
- **4つの運勢**: 大吉、中吉、小吉、凶の4種類からランダムに結果が表示されます。
- **レスポンシブデザイン**: ブラウザ上で動作し、ポップなカラー（黄色と赤）を基調とした親しみやすいデザインです。

## 使用技術

- **HTML5**: アプリケーションの構造
- **CSS3**: スタイリング、ボタンのホバーエフェクト
- **JavaScript (ES6)**: おみくじの抽選ロジック
